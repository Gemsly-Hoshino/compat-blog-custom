# 零成本建站指南：二级域名 + Cloudflare

## 二级域名 + Cloudflare
- **完全免费**：二级域名和 Cloudflare 托管均无费用，适合个人/学生党
- **全球加速**：Cloudflare CDN 可免费加速，中国大陆延迟不理想，可考虑国内CDN（使用国内 CDN 和服务器需要备案）。


## 申请免费二级域名
- **DigitalPlat**：提供 `.dpdns.org` 后缀，支持自定义前缀，半年要续费一次。
- **EU.ORG**：可申请eu.org后缀，目前似乎失效。
- **Cloudns**：DNS服务商，免费二级域名，风控较严格。

#### 查询方法（以 DigitalPlat 为例）

1.在首页输入你想要的域名，有xx.kg、us.kg 、dpdns.org、qzz.io四种域名可供选择，后两者是免费的

![打开 DigitalPlat 官网](4eef4bfa-32f6-42df-9ea0-59916b3314fe.png)

2.当网站显示，绿色的 **This domain name is now available**就说明这个域名已经可以注册了

![显示绿色可注册](aedd78c0-dd76-4da3-be97-27dfe41d7719.png)

#### 注册方法

1.回到网站的首页，点击注册一个域名，跳转到登录页面，是有 Google 登录和注册两个选项供新手选择的，这里我们选注册（Sign up）。

![登录页](QQ20251001-101632.png)

2.这里注册表单用户名和全名可以按照自己的喜好来填，这个用户名是不能重复的， Email就填自己的邮箱。电话号码和地址可以随便写一个，密码就填你自己的密码。它的要求是一个大写字符，一个小写字符和一个数字。全部填写好，没有红字警告之后点注册

![填写注册信息](3c639039-cdc6-40e4-9b23-4653583e16d6.png)

3.出现 success 画面之后，去邮件收件箱检查一下邮件

![显示邮件发送成功](QQ20251001-102250.png)

4.在邮件收件箱里找到刚刚发来的邮件，点蓝字链接验证下，验证过出现 successful 字样

![收到邮件点击邮件链接](QQ20251001-102353.png)

5.回到刚才的登录页，输入自己的邮箱和密码登录一下，新账户首次登录有一个 KYC 步骤，这里选择Github OAuth，登录一下自己的Github账户就可以了，没有就注册一个（点击下方的 创建账户）这里演示注册步骤。

![注册GitHub账号](QQ20251001-103034.png)

6.表单全部打勾之后，点创建账户。通过人机验证。

7.把发到邮箱的验证码输入一下

![输入邮箱收到的验证码](QQ20251001-103552.png)

8.重复授权页操作，登录一下你的账户。

![登录 GitHub 账户](QQ20251001-103848.png)

9.有时候授权页点击之后，不会跳转，手动回到控制台页面检查一下是否已经完成验证了

![授权 GitHub 账户](QQ20251001-103859.png)

10.出现控制台页即完成注册。

![成功出现控制台](QQ20251001-104117.png)

#### 绑定域名

1.Cloudflare官网点击登录，没有账号可以点底下 Sign up，页面右上角是有中文选项的，输入自己的邮箱和想要的密码。

![点击右上角登录或者注册账号](QQ20251001-104518.png)

2.新用户打开就是控制台的页面

![Cloudflare 账号仪表盘](QQ20251001-105046.png)

3.回到 [DigitalPlat 官网](https://DigitalPlat.org/)，点击域名注册（Domain Registration），同意协议，输入你想要的前缀（如 `john`）。有`dpdns.org`和`qzz.io`是免费的。两个 kg 后缀需要收费。
![DP](27583ce8-c708-4c05-b47f-258225c1b44e.png)

3. 显示绿色的可注册标签之后，回到 CloudFlare![显示可注册](QQ20251001-105227.png)

4.在Cloudflare控制台中添加域名，配置默认，输入现有域就输入刚刚自己在免费域名控制台注册的二级域名。
![添加域名](07745fa2-bbb5-4b5e-8ea5-30019f596a6c.png)

5.添加完成后，点击域名概述。添加名称服务器到NS1和NS2，点击注册（Register）。
![设置名称服务器](c70292ba-7bb8-40aa-82a7-c84c9c41b984.png)

1. 等待Cloudflare发送邮件，即完成激活。
    ![等待激活邮件](fe136672-3541-411d-be7a-a9524bdfa0d8.png)


## Cloudflare 配置与静态托管
1. 登录 Cloudflare，进入刚才注册的二级域名，点击侧边栏中的Worker路由-管理Worker-创建应用程序-Pages，随便输入一个名字，上传包含你的`index.html`的HTML文件夹，点击“部署”。
![上传项目文件](db08b9ad-04e9-48ba-976e-bb5a23caeb38.png)
2. 在Worker管理页面，选择自定义域，设置自定义域，输入注册并激活的二级域名，点击继续，完成。
3. 等待自定义域名显示活动，即完成配置。

## 效果验证与优势
- **访问测试**：打开你的二级域名（如 `john.dpdns.org`），确认页面正常加载且显示 HTTPS。
![网站成功打开](fdcad72d-445e-4ea8-a544-0683d805aa95.png)
- **成本统计**：全程无任何费用，域名、托管、CDN、SSL 全部免费。


## 拓展玩法
- 绑定多个二级域名，实现“多域名访问同一网站”。
- 利用 Cloudflare Workers 实现简单的动态功能（如表单提交、访问统计）。
- 使用Hugo搭建静态博客，可实现资源利用最大化。
